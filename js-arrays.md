# JavaScript Arrays (Backend Notes)

## Add / Remove
- push → add end
- pop → remove end
- shift → remove start
- unshift → add start

## Search
- includes → existence check
- indexOf → position

## Copy vs Modify
- slice → copy
- splice → modifies original

## Merge
- spread (...) → merge arrays
- concat → merge 2 arrays

## Convert
- Array.from → convert iterable
- Array.isArray → check array

## Backend Usage
- Pagination → slice
- Remove unwanted → filter
- Transform response → map
